# Amine S. Portfolio

Welcome to my personal portfolio website. This is a fully static website built with **HTML**, **CSS**, and **Bootstrap 5**, showcasing my professional experience and projects as a **Team Lead PHP & TYPO3 Developer**.

## Live Demo

Check out the live website here:  
[https://Amine1414.github.io/amines-portfolio/](https://Amine1414.github.io/amines-portfolio/)

---

## Features

- Fully responsive and mobile-friendly design using **Bootstrap 5**.
- Home page with a professional introduction and featured projects.
- About page highlighting experience, skills, and expertise.
- Projects page showcasing notable TYPO3 projects:
  - Emoji Selector – Custom CKEditor plugin for TYPO3 13
  - Migration Tool – CLI export/import for TYPO3 data
  - Anuaries Import – XML to TYPO3 importer
- Contact page with email, LinkedIn, and GitHub links.
- Consistent Navbar and Footer across all pages.
- Easy to customize and extend for future projects.
